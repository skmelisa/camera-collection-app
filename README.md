#Camera Collection App

React ve Tailwind CSS kullanılarak geliştirilmiş, kameraları ekleyip yönetebileceğiniz basit bir CRUD uygulaması.

Uygulama sayesinde kullanıcılar kamera koleksiyonlarını ekleyebilir, düzenleyebilir, silebilir ve arayabilir. Veriler **LocalStorage** içinde saklanır.

---

# 🚀 Özellikler

✅ Kamera ekleme
✅ Kamera düzenleme
✅ Kamera silme
✅ Marka ve model ile arama
✅ Görsel URL desteği
✅ Varsayılan placeholder görsel
✅ LocalStorage ile veri saklama
✅ Tailwind CSS ile modern tasarım
✅ Responsive arayüz

---

# 🛠 Kullanılan Teknolojiler

* React
* Tailwind CSS
* JavaScript
* LocalStorage
* Vite

# 📦 Kurulum

Projeyi çalıştırmak için:

```bash
npm install
npm run dev
```

Ardından tarayıcıda:

```
http://localhost:5173
```

# 🌐 Canlı Demo

Netlify üzerinden yayınlanabilir.

Deploy adımları:

1. GitHub'a push et
2. Netlify → **New site from Git**
3. GitHub repository seç
4. Build command:

```
npm run build
```

Publish directory:

```
dist
```
